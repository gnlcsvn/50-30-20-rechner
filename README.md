# 50/30/20 Finanzrechner

Ein interaktiver Finanzrechner, der die 50/30/20-Regel zur Budgetplanung nutzt und zusätzlich die Vermögensentwicklung durch Investitionen visualisiert.

## Features

- Berechnung der Budgetverteilung nach der 50/30/20-Regel
- Investitionsrechner mit:
  - Einstellbarer Rendite
  - Wählbarem Anlagehorizont
  - Visualisierung der Vermögensentwicklung
  - Berechnung von Zinserträgen und Endkapital
